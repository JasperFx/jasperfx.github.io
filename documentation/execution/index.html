Execution Pipeline

Jasper's execution pipeline for handling messages is unique. Instead of forcing you to constrain your code to meet Jasper's interfaces, Jasper tries to adapt itself to your code. It does this by taking your Writing Message Handlers and whatever Jasper Middleware and Policies, then generating C# code to call your code in the most efficient way possible using Roslyn's ability to compile code at runtime.
