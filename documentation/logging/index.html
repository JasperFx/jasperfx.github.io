Logging Integration

The Jasper messaging is instrumented with the ASP.Net Core ILogger abstraction and infrastructure throughout now. The logging would be configured just like an ASP.Net Core application using JasperRegistry.Hosting.ConfigureLogging() or the normal IWebHostBuilder.ConfigureLogging() method you may already be using.

There are two subjects:

  1. Jasper.Transports - information about Jasper's running transports, including circuit breaker events and messaging failures
  2. Jasper.Messages - information about specific messages
